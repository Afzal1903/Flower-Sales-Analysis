# 🌸 Flower Sales Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing sales data from a flower and gifting business. The goal is to extract meaningful insights related to customer behavior, product performance, seasonal trends, and revenue generation using Excel-based data analysis.

The dataset includes information about:

* Customers
* Orders
* Products
* Revenue and sales patterns

## 🧰 Tools & Technologies Used

* Microsoft Excel
* Data Cleaning & Transformation
* Pivot Tables & Charts
* Basic Data Analysis Techniques

## 📂 Dataset Description

### 1. Customer Table

Contains customer-related details:

* Customer ID
* Name
* City
* Gender
* Contact Details

### 2. Orders Table

Tracks transaction-level data:

* Order ID
* Customer ID
* Order Date
* Product Details
* Revenue
* Day of Order

### 3. Product Table

Provides product-level information:

* Product ID
* Product Name
* Category (Soft Toys, Plants, Sweets, etc.)
* Price
* Occasion (Diwali, Valentine’s Day, All Occasions)

## 📊 Key Analysis Performed

* Revenue analysis across different occasions
* Sales trends by day of the week
* Product category performance
* Customer segmentation by location and gender
* Identification of high-performing products

## 📈 Dashboard

An interactive Excel dashboard was created to visualize:

* Total revenue
* Sales by category
* Occasion-wise sales distribution
* Day-wise order trends

## 🎯 Objectives

* Understand customer buying behavior
* Identify peak sales periods
* Optimize product offerings
* Improve business decision-making through data

## 🚀 Key Takeaways

* Certain occasions (like Valentine’s Day and Diwali) drive major revenue spikes
* Specific product categories consistently outperform others
* Sales vary significantly depending on the day of the week

## 📌 Future Improvements

* Integration with Power BI for advanced visualization
* Predictive analysis using Python
* Customer lifetime value analysis
* Recommendation system for personalized offers

---

This project demonstrates how raw sales data can be transformed into actionable business insights using simple tools like Excel.
